**Description:**

As an HR administrator, I need a comprehensive form to add new employees to the system with all their personal details, employment information, bank details, and emergency contacts in a single submission.

**User Story:**

As an admin, I want to click an "Add Employee" button and fill out a form with all employee details so that I can onboard new team members efficiently without making multiple requests.

**Success Flow:**
1. User clicks "Add Employee"
2. Form opens with all fields empty
3. User fills required fields (firstName, lastName, email, jobTitle)
4. User optionally fills other fields
5. User clicks "Add Employee" button
6. Form validates required fields
7. POST request sent to /employees
8. Success alert shown
9. Form closes and clears
10. Employee list refreshes with new employee

**Error Handling:**
- [ ] Network error: Show "Failed to save employee: [error]"
- [ ] Validation error: Show specific field errors
- [ ] Duplicate email: Show "Employee with this email already exists"
- [ ] Any error displays in alert dialog
